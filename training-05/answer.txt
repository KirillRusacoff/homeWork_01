1. Какие бывают алгоритмы сортировок?

Ответ:

1. Bubble Sort - алгоритм проходит по массиву несколько раз. На каждом проходе он сравнивает пары соседних элементов и меняет их местами, если они находятся в неправильном порядке.
- Прост в реализации.
- Неэффективен для больших массивов из-за квадратичной временной сложности.

2. Selection Sort - алгоритм разделяет массив на две части: отсортированную и неотсортированную. На каждом шаге выбирается минимальный элемент из неотсортированной части и меняется местами с первым элементом в неотсортированной части.
- Прост в реализации.
- Выполняет фиксированное количество обменов на каждом шаге.

3. Insertion Sort - алгоритм строит отсортированную часть массива по одному элементу за раз. На каждом шаге выбирается элемент из неотсортированной части и вставляется в правильное место в отсортированной части.
- Эффективен для небольших массивов или частично отсортированных данных.
- Вставка элементов в отсортированную часть может быть затратной для больших массивов.

4. Merge Sort - алгоритм использует принцип "разделяй и властвуй". Массив рекурсивно разделяется пополам, сортируются две половины, и затем результаты объединяются.
- Обеспечивает стабильную сортировку.
- Эффективен для больших массивов, но требует дополнительной памяти для временных массивов.

5. Quick Sort -  алгоритм использует принцип "разделяй и властвуй", разделяя массив на подмассивы вокруг опорного элемента, который затем устанавливается в правильное место. Процесс повторяется рекурсивно.
- Один из самых эффективных алгоритмов сортировки.
- Не требует дополнительной памяти для временных массивов (in-place).
